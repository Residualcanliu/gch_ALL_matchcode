# AI智能眼镜多模态功能技术文档

> 最新讨论和与黄老师商讨结果后，手势识别将改为**时序模型**，而**路面障碍检测**将延续yolo8，所以下面的mediapipe库识别方法将精简且暂不写**扩展方面**章节

## 一、功能概述
多模态由两大核心功能组成，分别为**手势识别**与**路面障碍检测**两大核心功能，通过摄像头实时采集图像，利用**Mediapipe库**实现手势（OK、2、7 等）的精准识别。通过**YOLOv8**模型检测路面障碍物（行人、车辆、自行车等），可扩展支持手势控制外卖接单、路面危险提醒等场景，为用户提供便捷的交互与环境感知能力。

## 三、主要功能介绍
### 1.手势识别：
通过摄像头实时检测单手手势（默认为右手），同时具备防抖功能，检测如下手势触发相关功能：  
- ***“OK手势”***：暂为输出“OK”，后续可以修改为外卖接单等
- ***“2手势”（食指+中指伸直）***：暂为输出“2”
- ***“7”手势（拇指+食指伸直）***：暂为输出“7”
### 2.路面障碍检测：
通过摄像头实时监测路面情况，识别路面常见障碍物，实时输出检测结果并生成警告，检测物品暂时如下：
- ***"person"***: "行人",
- ***"car"***: "车辆",
- ***"truck"***: "卡车",
- ***"bicycle"***: "自行车",
- ***"motorbike"***: "摩托车",
- ***"dog"***: "动物",
- ***"cat"***: "动物"

> 主要功能存在可扩展性，详细可以查看“扩展方面”章节与“微调方面”章节

## 四、详细参数讲解
### 1.手势识别模块
- GestureConfig：配置参数类，可设置摄像头索引、帧率（默认 20FPS）、图像分辨率（默认 640×480）、手势冷却时间（默认 1 秒，防止抖动）等。
- HandGestureDetector：手势检测核心类，包含：
    - 初始化 Mediapipe 手部检测模型
    （init_mediapipe）
    - 摄像头（init_camera）
    - 实时处理图像帧（process_gestures）：镜像翻转、提取右手关键点、调用分类器识别手势。
    - 手势防抖：（gesture_cooldown）控制检测间隔，避免短时间内重复触发。

- is_finger_straight / is_finger_bent：通过指尖与掌指关节的Y坐标判断手指状态。
- 预定义手势：
    - OK 手势：拇指与食指尖端距离＜0.05，且中指、无名指、小指伸直。
    - 2 手势：食指、中指伸直，其余手指弯曲。
    - 7 手势：拇指、食指伸直，其余手指弯曲。

### 2.路面障碍检测模块
- Config：配置参数类，包含yolov8n.pt模型路径、目标检测类别（行人、车辆等）、镜像翻转开关等。
- RoadObstacleDetector：障碍检测核心类，包含：
    - WarningGenerator：警告生成，通过输出障碍物提示（如 “小心行人！”）。
    - __init__：初始化检测器，接收配置参数，初始化模型、摄像头等属性，打印镜像配置状态  
    - load_model：加载YOLO模型，成功返回True，失败返回False并记录错误日志
    - init_camera：初始化摄像头，打开指定索引的摄像头，成功返回True，失败返回False并记录错误日志
    - _mirror_frame：私有方法，根据配置对图像帧进行镜像翻转处理（水平翻转）
    - process_detection：处理单帧图像检测，先进行镜像处理，再执行YOLO检测，提取并返回含检测框的图像帧及检测到的目标列表，筛选置信度＞0.5的目标
    - run_detection：运行实时检测主循环，循环读取摄像头帧，处理检测，输出提示信息，显示图像，响应退出按键
    - _print_warnings：私有方法，对检测到的目标输出提示信息（如“小心行人!”）
    - release_resources：释放摄像头资源，关闭所有显示窗口，停止检测运行

## 五、使用方法
### 1.手势识别模块
- 程序启动后自动打开摄像头，默认显示 “Waiting for gesture...”。
- 伸出右手做出 OK、2、7 手势，画面将显示识别结果（如 “Gesture: ok”）。
- 按q键退出程序。

### 2.路面障碍检测模块
- 需提前将yolov8n.pt模型下载好并放入根目录下，设置好模型路径后再启动
- 程序实时显示检测画面，框选障碍物并输出警告日志（如 “小心车辆！”）
- 按q键退出程序。

## 六、微调方面
### 1.手势识别模块微调
- **摄像头参数**：修改GestureConfig类的camera_index（摄像头索引）、frame_width/frame_height（分辨率）、fps（帧率）。
- **检测灵敏度**：调整detection_confidence（默认为 0.7）和tracking_confidence（默认为 0.6），提高值可降低误检但可能漏检。
- **防抖时间**：修改gesture_cooldown（默认 1.0 秒），缩短时间可提高响应速度，但可能增加抖动。

### 2.路面障碍检测模块微调
- **目标类别**：修改Config.TARGET_CLASSES，添加 / 删除需要检测的目标（如增加 “chair”：“椅子”）。
- **置信度阈值**：在process_detection函数中调整confidence > 0.5的阈值，提高值可减少误检。
- **模型路径**：修改Config.MODEL_PATH，更换为 YOLOv8 其他模型（如yolov8s.pt，精度更高但内存占用更大）。

## 八、注意事项
1.**硬件限制**：嵌入式设备需保证摄像头正常工作，内存≥1GB  
2.**模型文件**：路面检测模块依赖yolov8n.pt，需要确认是否有相关模型，要不然他会自动下载，特别慢。
3.**手势识别范围**：支持单手手势，需在摄像头可视范围内（建议距离30-50cm）  
4.**权限问题**：确保程序拥有摄像头访问权限

## 九、故障排除
| 问题现象 | 可能原因 | 解决方法 |
| :---- |:----:| :----: |
| 摄像头无法打开 | 索引错误或被占用 | 修改camera_index（尝试 0、1）；关闭其他占用程序 |
| 手势识别无响应 | 未检测到右手或光线不足 | 调整手部位置；增加环境亮度 |
| 障碍物检测卡顿 | 设备性能不足 | 降低分辨率（如 480×320） |
| 模型加载失败 | 路径错误或文件损坏 | 检查MODEL_PATH，或者没下载 |
| 频繁误识别手势 | 防抖时间过短或置信度低 | 增大gesture_cooldown；提高detection_confidence |

## 十、嵌入内存计算
### 1.Python解释器（约20-50MB）：
嵌入式 Python（如Python 3.11的ARM精简版）运行时占用约20-50MB，主要包括解释器核心、垃圾回收机制等。
### 2.核心依赖库（约200-330MB）：
- OpenCV（图像处理）：加载后占用约40-60MB（包含图像处理算法、矩阵运算模块）
- Mediapipe（手势检测）：轻量级手势模型（手部关键点检测）加载后约 30-50MB（模型参数 + 推理框架）。
- Ultralytics（YOLOv8）：YOLOv8-nano模型（yolov8n.pt）加载后约 80-120MB。
- Numpy（数值计算）：约10-20MB（数组运算核心）。
- 其他辅助库（logging、time等）：合计约10-20MB。
### 3.核心检测模块运行时内存（约80-150MB）：
两个核心功能（手势检测 + 路面障碍检测）的实时运行内存，主要来自图像数据和临时计算：  
1.**图像数据存储**：  
眼镜摄像头通常输出 640×480 分辨率的图像（与代码中frame_width=640, frame_height=480一致），RGB 格式下单帧图像大小为：  
`640×480×3字节（RGB）=921,600字节≈0.9MB`  
由于两个模块可共享同一帧图像（避免重复存储），实际占用约0.9MB。  
2.**手势检测模块**：
- Mediapipe 手部关键点检测时，需临时存储21个手部关键点坐标，每个点含 x/y/z 浮点数，共 21×3×8 字节 = 504 字节，可忽略。
- 推理过程中临时特征图和计算缓存：约5-10MB。  

3.**路面障碍检测模块**：
- YOLOv8-nano推理时，需存储中间特征图（卷积层输出）和检测框数据，nano模型的特征图尺寸较小，约 50-100MB。
- 检测结果（目标类别、坐标）存储：约0.1MB。 

4.**扩展功能内存**  
因为后续需要接入“手势控制外卖接单”和“路面提醒系统”的额外内存，大概为**50-100MB**，包括：
- 网络通信（如HTTP请求库requests）：加载后约 5-10MB，运行时临时缓存（请求/响应数据）约1-2MB
- 状态管理（如 “是否接单”、订单信息）：数据结构（字典/列表）约0.1MB。
- 路面提醒系统，市面常见模型+运行时约30-60MB  

**总体算下来，建议眼镜设备内存至少为1GB**
