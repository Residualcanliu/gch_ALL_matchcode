## 📖 项目简介

本项目为 **2026年中国大学生计算机设计大赛 — 大数据主题赛** 参赛作品集合，整合了多个相关竞赛与实训项目的代码成果，涵盖：

- 🕶️ **AI智能眼镜多模态交互** — 基于 MediaPipe + YOLOv8 的实时手势识别与路面障碍检测系统
- 📝 **社交媒体评论分析** — B站 / 抖音 / 微博评论爬取、情感分析、主题词典匹配与优先级建模
- 🎓 **VR校园与电商数据分析** — VR设备市场调研，电商 + 视频平台评论情感挖掘与需求主题提取
- 📊 **模型性能对比可视化** — CVIT 轻量模型与主流 CNN / Transformer 架构的参数量/准确率对比

> **所有比赛均已结束，本仓库代码仅供学习交流与个人公开参考。**

## 🕶️ AI智能眼镜多模态交互

> 📂 代码路径：`2026年计算机设计大赛大数据主题赛/25国创技术方面demo/`
> 
> 🏷️ 来源：2025年大学生创新创业训练计划（国创）

### 功能矩阵

| 模块 | 技术路线 | 核心能力 | 适用场景 |
|:---|:---|:---|:---|
| 🖐️ 手势识别 | MediaPipe + 自定义分类器 | 识别 OK / 2 / 7 等手势，防抖控制 | 外卖接单、无接触操控 |
| 🚗 障碍检测 | YOLOv8 (nano) | 实时检测行人/车辆/动物，生成警告 | 路面危险提醒、辅助驾驶 |

### 核心文件速览

| 文件 | 功能 |
|:---|:---|
| `基于mediapipe检测手势指令.py` | 手势识别（普通摄像头） |
| `基于mediapipe检测手势指令（眼镜摄像头）.py` | 手势识别（眼镜摄像头适配） |
| `基于yolov8检测路面障碍.py` | 障碍检测（普通摄像头） |
| `基于yolov8检测路面障碍（眼镜摄像头）.py` | 障碍检测（眼镜摄像头适配） |
| `技术文档.md` | 📄 完整文档：环境配置 / 参数详解 / 微调 / 故障排除 |

### 硬件要求

- 摄像头分辨率 ≥ 640×480
- 内存 ≥ 1GB（嵌入式 ARM 设备）
- 模型文件：首次运行自动下载 `yolov8n.pt`

> 📄 详细技术文档：[技术文档.md](2026年计算机设计大赛大数据主题赛/25国创技术方面demo/技术文档.md)

## 📈 模型对比可视化

> 📂 文件：`2026年计算机设计大赛大数据主题赛/test1.py`

CVIT 轻量模型与主流架构的性能对比：

| 模型 | 参数量 (M) | 准确率 (%) | 特点 |
|:---|:---|:---|:---|
| **CVIT模型** | **2.11** | **98.62** | 🏅 轻量 + 高精度 |
| CVIT（不带VIT） | 2.12 | 97.75 | |
| 传统CNN | 2.49 | 98.47 | |
| ViT-Base | 85.27 | 97.06 | 参数量大 |
| 通用CNN+Transformer | 90.39 | 98.75 | 参数量最大 |

> 💡 **结论**：CVIT 模型以仅 2.11M 的参数量达到 98.62% 准确率，在轻量化与精度之间取得最优平衡。
